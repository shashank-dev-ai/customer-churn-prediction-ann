# Customer Churn Prediction using ANN

## Overview
This project predicts whether a customer will leave a bank using an Artificial Neural Network (ANN).

## Dataset
- Customer Churn Dataset
- Features include:
  - Credit Score
  - Geography
  - Gender
  - Age
  - Balance
  - Estimated Salary
  - Tenure
  - Number of Products

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow
- Keras
- Matplotlib

## Workflow
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Feature Encoding
5. Feature Scaling
6. Train-Test Split
7. ANN Model Building
8. Model Training
9. Model Evaluation
10. Prediction

## Model
- Input Layer
- Hidden Layers (ReLU)
- Output Layer (Sigmoid)

## Results
- Binary classification for customer churn prediction.
